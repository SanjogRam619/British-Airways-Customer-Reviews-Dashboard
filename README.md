✈️ British Airways Customer Reviews Dashboard

An interactive Tableau dashboard developed to analyze British Airways customer reviews and transform raw customer feedback into meaningful business insights through data visualization.

---

## 📌 Project Overview

This project focuses on analyzing customer reviews of British Airways using Tableau Public. The dashboard provides an interactive view of passenger satisfaction by visualizing service quality, travel preferences, recommendation trends, and overall customer experience.

The objective is to help stakeholders understand customer feedback more effectively and support data-driven decision-making through intuitive visualizations and interactive filters.

---

## 🎯 Project Objectives

- Analyze overall customer satisfaction.
- Compare ratings across different countries.
- Identify monthly trends in customer reviews.
- Evaluate service quality across multiple parameters.
- Analyze customer satisfaction by traveller type and seat type.
- Identify the top-rated aircraft based on customer reviews.
- Provide an interactive dashboard for business analysis.

## 🔎 Dashboard Filters

The dashboard allows users to interactively filter the analysis by:

- Country
- Traveller Type
- Seat Type
- Month

These filters dynamically update all visualizations, enabling users to explore customer feedback from different perspectives.

---

## 💡 Key Business Insights

- Customer satisfaction varies across different countries.
- Premium travel classes generally receive higher ratings.
- Customer preferences differ among traveller types.
- Some aircraft consistently receive better customer ratings.
- Monthly trends help identify changes in customer satisfaction over time.
- Recommendation rate reflects overall customer loyalty and satisfaction.
